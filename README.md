# Nyaa Anime BR

Nyaa Anime BR é um addon para o Stremio que busca torrents do **Nyaa.si** contendo **legendas em português** para animes.

Este repositório **não contém o código-fonte do addon**. Ele existe apenas para centralizar o registro de bugs, sugestões e discussões da comunidade.

## Addon

https://stremio-addons.net/addons/nyaa-anime-br


## O que o addon faz

- Pesquisa torrents no Nyaa.si.
- Filtra lançamentos que possuem legendas em português.
- Disponibiliza os resultados diretamente no Stremio.

## Reportando problemas

Se você encontrou algum problema, abra uma **Issue** informando, se possível:

- Nome do anime.
- Episódio ou temporada.
- O que aconteceu

## Sugestões

Novas ideias e melhorias são bem-vindas. Utilize a aba **Issues** para enviar sugestões.

## Aviso

Este projeto apenas indexa informações disponíveis publicamente e **não hospeda, distribui ou armazena arquivos de mídia**. O addon apenas retorna metadados e links magnet obtidos de fontes públicas.
